Timestamp Microservice

Convert between Unix timestamps and UTC dates

About This Service

This API service converts dates between Unix timestamps and UTC strings. It's a solution for the FreeCodeCamp Back End Development and APIs certification project.

API Usage Examples

Use these endpoints to get timestamp conversions:

Valid Date String

/api/2015-12-25

Returns: {"unix":1451001600000,"utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

Unix Timestamp

/api/1451001600000

Returns: {"unix":1451001600000,"utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

Current Time

/api

Returns the current time in both Unix and UTC format
